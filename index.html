Know Your Location Trail

So, Whats location trail?

Location trail refers to a time stamped list of locations of a device which can be found using the GPS.

This is open source project to demonstate how the location traling works, you can take a look at source code by visiting repo hosted on github.

Press this button to begin locating your location.

Note: It require GPS to track location, So make sure your location services are turned on.

Note: This only shows latitude and longitude(latlng) because for exact address I'd need to implement some proprietary third party api which will convert latlng to plain address And i dont want to send your coordinates to them because "I care about your privacy!". But if you want you can do it yourself. For that, one of many sites is https://www.latlong.net/Show-Latitude-Longitude.html
